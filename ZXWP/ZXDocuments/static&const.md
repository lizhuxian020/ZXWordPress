# const

```c
int i = 1;
int j = 3;

const int i = 2;
int const i = 2;
/*
上述 2 个效果一样, 都是 i 不能变
*/

int const *a = &i;
const int *a = &i;
/*
上述两个效果一样, 都是(*i)不能变:
1. 在栈声明了一个地址空间(8 字节), 指向 int 整形, "指向的空间"不能变!
2. 但堆的地址空间可以变, 即: a = &j; 是允许的
*/

int* const a = &i;
/*
上述是在堆里声明了一个地址空间(8 字节), 这个堆空间内容不能变, 但堆里地址指向的空间可变: *a = 4;是可以的
*/


```
